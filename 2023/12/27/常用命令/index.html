因项目添加了子模块https://github.com/jerryc127/hexo-theme-butterfly.git

在clone 项目时使用：

git clone --recursive https://github.com/zrmomo/hexo-blog.git

一次性拉下来两个模块

更新子模块：

在根项目目录下

更新子模块到最新版本：git submodule update

更新子模块为远程项目的最新版本：git submodule update --remote

电脑下载hexo

在空文件夹初始化Hexo

进入然后输入

借鉴文章：

  • https://iphysresearch.github.io/blog/post/programing/git/git_submodule/

  • https://juejin.cn/post/7237366474472783931?searchId=20231227212646706C7C7C04BD8F06F29D

Hexo指令：https://hexo.io/zh-cn/docs/commands

生成静态文件：

hexo g

本地预览：

hexo s

部署网站

hexo d
